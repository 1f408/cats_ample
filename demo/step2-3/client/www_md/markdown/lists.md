# リスト(箇条書き)

## 番号無しリスト

``` md
* item-1
  * sub-item-1
  * sub-item-2
- item-2
  - sub-item-3
  - sub-item-4
+ item-3
  + sub-item-5
  + sub-item-6

```
* item-1
  * sub-item-1
  * sub-item-2
- item-2
  - sub-item-3
  - sub-item-4
+ item-3
  + sub-item-5
  + sub-item-6

## 番号ありリスト

``` md
1. item-1
   1. sub-item-1
   2. sub-item-2
2. item-2
 1. sub-item-3
 2. sub-item-4
3. item-3

```

   2. sub-item-2
